# Multi-Step Onboarding

## Requirements
- save every step locally
- restore progress
- minimal UI
- selectable inputs only

## Example Steps

Step 1:
- age group

Step 2:
- cycle length

Step 3:
- symptoms

Step 4:
- goals

Step 5:
- pregnancy status

## Important Rule
After every step:
- save progress
- save current step

If app closes:
- continue from saved step
